Visão Geral

O FPS Launcher App Site é um site institucional cujo objetivo é servir como ponto de entrada para os usuários do FPS Launcher, fornecendo informações claras sobre o launcher, seus recursos, como instalá-lo, além de apresentar a comunidade por trás do projeto. A ideia é unir uma presença web profissional com uma interface fácil para novos usuários conhecerem o launcher.

Este projeto demonstra também minhas habilidades em design front-end, desenvolvimento web e criação de sites estáticos / dinâmicos (dependendo da implementação), sendo ideal para portfólio, apresentação para stakeholders e publicidade.

# Funcionalidades

Página inicial com apresentação do launcher: o que é, para quem serve, principais vantagens.

Seção de “Como usar”: guia para instalar e configurar o launcher.

Capturas de tela ou ilustrações (ex: imagens de personagens, ícones) para dar vida visual.

Página de doações (“Doação”): facilita o apoio da comunidade ou monetização.

Página de instalador (“Installer”): link para instalar ou baixar o launcher.

Ícones personalizados (ex: icone-fps.ico) para reforçar a identidade visual.

Recursos estáticos: site simples, leve e otimizado para performance.

Instalação e Execução

Como este é um site estático (ou semi-estático), o processo de “instalação” é simples. Abaixo está um guia para executar localmente:



 


Navegue até a pasta do projeto:

cd fpslauncherappsite  


Abra o arquivo index.html no seu navegador (duplo clique ou via servidor local):

Opção A (abertura simples):
Abra diretamente o index.html no seu navegador.

Opção B (via servidor local):
Se preferir usar um servidor local (recomendado para desenvolvimento):

# Com Python 3.x instalado
python -m http.server 8000


Depois, abra http://localhost:8000 no navegador.

Deploy / Hospedagem

Você pode fazer deploy do site utilizando diversas opções:

GitHub Pages: uma das soluções mais simples, diretamente do branch main (ou outro branch configurado).

Netlify / Vercel: para hospedagem gratuita com SSL automático.

Servidor próprio: caso você tenha um domínio ou servidor, pode fazer upload dos arquivos estáticos.
